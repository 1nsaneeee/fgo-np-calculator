# FGO NP Damage Calculator // 宝具伤害计算器

Fate/Grand Order 宝具伤害与连发回收计算器，单页静态应用，浏览器直接打开即用。

## 功能

- **宝具伤害计算** — 支持 NP1~NP5、等级 1~120、芙芙/礼装 ATK 加成、阵营克制
- **完整 Buff 面板** — CE/自身/辅助/敌方/Debuff 五类 Buff，魔放/宝威/攻击力/特攻全覆盖
- **指令卡链模拟** — 点击切换 Buster/Arts/Quick 三色卡牌，实时计算总伤害/NP回收/暴击星
- **3T 连发模拟** — 三回合 NP 回收预测，判断能否宝具连发
- **自定义从者** — 自由设置职阶、配卡、Hit 数、NP率、宝具倍率，完全定制
- **435 从者数据库** — 内置完整从者数据，支持职阶/关键词搜索
- **多敌配置** — 敌方可选职阶/阵营/防御力，支持鞭尸和暴击判定

## 用法

用浏览器打开 `FGO-Calculator.html` 即可：

1. 顶部搜索框查找从者，或点击**自定义**自行设定参数
2. 左侧设置等级、宝具等级、Buff 数值
3. 右侧实时显示宝具伤害计算结果
4. 切换暴击/鞭尸状态查看对 NP 回收的影响
5. 拖拽指令卡链模拟普通指令卡输出

## 部署

纯静态 HTML 文件，托管到任意静态服务即可：

```bash
# 本地预览
python -m http.server 8080
# 或直接浏览器打开 FGO-Calculator.html
```
